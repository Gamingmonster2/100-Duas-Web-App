## 🛠️ المميزات التقنية (Technical Features)
- **Responsive UI**: واجهة متجاوبة تعمل بامتياز على الجوال والحاسوب.
- **Fast Loading**: تحسين الصور لضمان سرعة تحميل فائقة.
- **Navigation**: أزرار تحكم سهلة للتنقل بين الأدعية (السابق والتالي).
- **No Backend**: تطبيق خفيف يعمل بالكامل على المتصفح (Client-side).

---

## 📂 هيكل الملفات (Structure)
- `index.html`: هيكل التطبيق الرئيسي.
- `style.css`: التصميم والألوان العصرية.
- `script.js`: منطق التنقل بين الصور.
- `/images`: المجلد المحتوي على صور الأدعية الـ ١٠٠.
